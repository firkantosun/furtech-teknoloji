# Project images

Card thumbnails live in this folder. Each file maps to a project card on the homepage and `/projects`.

| File | Service area |
|------|----------------|
| `web-development.svg` | Web Sitesi Geliştirme |
| `mobile-app.svg` | Mobil Uygulama Geliştirme |
| `bodrum-aquapark.svg` | Bodrum Aquapark Dijital Sistem |
| `management-panel.svg` | Yönetim Paneli ve Operasyon Yazılımları |

**Replace with your own photos:** use the same filenames with `.webp` and update `image` paths in `lib/data.ts`.

**Recommended specs for custom photos:** 1600 × 1000 px · 16:10 · WebP · under 400 KB
